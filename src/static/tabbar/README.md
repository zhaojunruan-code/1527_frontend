# Tabbar 图标说明

本目录存放底部导航栏的图标文件。

## 医生角色图标（3个tab）
- `doctor-home.png` - 医生首页（未激活）
- `doctor-home-active.png` - 医生首页（激活）
- `message.png` - 消息（未激活）
- `message-active.png` - 消息（激活）
- `mine.png` - 我的（未激活）
- `mine-active.png` - 我的（激活）

## 患者角色图标（4个tab）
- `patient-home.png` - 患者首页（未激活）
- `patient-home-active.png` - 患者首页（激活）
- `doctor.png` - 医生（未激活）
- `doctor-active.png` - 医生（激活）
- `message.png` - 消息（未激活，与医生共用）
- `message-active.png` - 消息（激活，与医生共用）
- `mine.png` - 我的（未激活，与医生共用）
- `mine-active.png` - 我的（激活，与医生共用）

## 图标规格
- 尺寸：建议 64x64 px 或 128x128 px
- 格式：PNG（支持透明背景）
- 颜色：未激活状态建议灰色，激活状态建议主题色
